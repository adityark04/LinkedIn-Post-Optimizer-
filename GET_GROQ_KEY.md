## Troubleshooting

### "Groq API key not found"
Make sure you set it in PowerShell:
```powershell
$env:GROQ_API_KEY = "gsk_your-actual-key"
```

Check it's set:
```powershell
echo $env:GROQ_API_KEY
```

### "ModuleNotFoundError: No module named 'groq'"
```powershell
pip install groq
```

### Backend still shows "Falling back to pattern-based"
1. Kill any running Python processes
2. Set the GROQ_API_KEY again
3. Restart backend
